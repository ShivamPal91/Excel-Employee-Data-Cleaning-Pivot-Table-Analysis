# Excel Employee Data Cleaning & Pivot Table Analysis

This project demonstrates an end-to-end data preparation and analysis workflow using Microsoft Excel. The primary objective was to clean a raw employee dataset, improve its quality, and perform exploratory analysis using Pivot Tables to generate meaningful business insights.

The project highlights practical Excel skills commonly required for Data Analyst roles, including data cleaning, preprocessing, summarization, and reporting.

# Project Objectives
Clean and organize raw employee data.
Improve data quality by removing inconsistencies.
Prepare the dataset for analysis.
Create Pivot Tables to summarize key business metrics.
Generate actionable insights from the cleaned data.

# Tools & Technologies
Microsoft Excel
Pivot Tables

Sort & Filter
Remove Duplicates
Data Validation
Conditional Formatting

# Employee Sample Dataset

The dataset contains employee-related information such as employee details, department, age, salary, gender, location, and other business attributes.

# Data Cleaning Process

The following data preparation steps were performed:

Removed duplicate records
Eliminated extra spaces using Excel text functions
Deleted blank rows and empty columns
Corrected inconsistent text formatting
Standardized data values
Improved data accuracy and consistency
Validated records for analysis readiness

# Pivot Table Analysis

After cleaning the dataset, Pivot Tables were created to analyze business performance from different perspectives.

Examples include:

Employee count by department
Gender distribution
Average salary by department
Employee distribution by location
Age group analysis
Summary statistics for business reporting

These Pivot Tables transformed raw data into easy-to-understand summaries, enabling faster decision-making.
